### Коротко

Генерация кода Playwright по прогону прямо в интерфейсе: можно следить за ходом работы и смотреть готовый сценарий. Упростили запуск тест-кейсов и экран прогона, добавили демонстрационные примеры.

### Тест-кейсы и код

-  Добавили вкладку **Code** у автоматических тест-кейсов: после успешной генерации можно открыть и скопировать готовый сценарий Playwright.
-  Добавили **генерацию кода по прогону**: запуск из карточки прогона, статус задачи, подробный журнал и удобные режимы панели (**Expand code**, **Expand log** или оба блока рядом через компактный вид).
-  Сделали **подсветку фрагмента кода**, связанного с выбранным шагом прогона, чтобы проще сопоставлять шаги и строки сценария.
-  Показали понятные подсказки, если генерация недоступна (например, для ручных кейсов или когда нужен новый прогон после изменений).

### Запуск прогона

-  Сделали **выбор способа запуска** там, где это поддерживается: привычный прогон по шагам сценария или прогон по уже сгенерированному коду Playwright — через меню у кнопки запуска.

### Экран прогона

-  Встроили **панель генерации кода** в процесс прогона: можно следить за ходом генерации и переключать удобный вид без ухода со страницы.
-  Улучшили отображение **шагов и деталей** во время выполнения, чтобы проще ориентироваться в ходе прогона.
